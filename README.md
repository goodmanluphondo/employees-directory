# Employees Directory

## Installation

1. Clone the repository:

```bash
git clone git@github.com:goodmanluphondo/react-app.git
```

2. Navigate to the project directory:

```bash
cd react-app
```

3. Install dependencies:

```bash
npm install
```

## Running the application

Start the development server:

```bash
npm run dev
```
